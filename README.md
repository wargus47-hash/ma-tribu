# Ma Tribu 🏠

Application web personnelle (PWA) d'organisation familiale, pensée pour un parent en garde alternée.

- 🛒 **Courses** rangées par rayon + produits récurrents + budget
- 🍽️ **Repas** de la semaine + recettes qui génèrent la liste de courses
- 📅 **Garde** : calendrier de présence + 🎒 sac de transition entre les deux maisons
- 🧸 **Famille** : routines par enfant (mode enfant à pictogrammes), rappels, pense-bête

100 % **hors-ligne**, sans compte, **données stockées sur l'appareil** (rien n'est envoyé en ligne).

## Utiliser
Ouvrir `index.html` dans un navigateur, ou la version publiée (GitHub Pages).
Sur mobile : ouvrir le lien puis « Ajouter à l'écran d'accueil ».
